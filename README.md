# AI Text Summarizer

This project is a simple AI-powered text summarization web app built using:

- Streamlit
- LangChain
- HuggingFace LLM

## Features
- Enter long text
- Generate short summary
- Uses HuggingFace LLM via LangChain

## Tech Stack
Python, Streamlit, LangChain, HuggingFace

## Run Locally

Create virtual environment

python -m venv venv

Activate environment

venv\Scripts\activate

Install dependencies

pip install -r requirements.txt

Run the app

streamlit run app.py
